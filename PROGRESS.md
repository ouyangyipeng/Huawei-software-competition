# 华为软件精英挑战赛 - 进度记录

## 比赛概述

### 题目名称
简单多边形重叠消除 - 最小平移向量计算

### 核心问题
给定两个简单多边形A和B，计算最小平移向量(MTV, Minimum Translation Vector)，使多边形B沿该向量平移后与多边形A完全无重叠。

### 应用场景
二维排样、工业制造中的材料利用率优化

---

## 关键技术要点

### 1. 输入输出格式
```
输入：
- 第1行: n1 n2 (多边形A和B的顶点数)
- 第2行: 多边形A的顶点坐标 (2*n1个浮点数，逆时针)
- 第3行: 多边形B的顶点坐标 (2*n2个浮点数，逆时针)
- OK
- 第4行: K (测试样本点数量)
- 后续K行: 每行一个移动向量(x, y)
- OK

输出：
- K行最小平移向量
- OK
```

### 2. 时间限制
- 预处理：1秒
- 求解：10分钟（所有测试样本点）

### 3. 精度要求
- 阈值：ε = 1e-4
- 输出保留至少5位小数

### 4. 环境限制
- Ubuntu 22.04, 1核CPU, 2GB内存
- 单文件编译
- Python只支持numpy

## 算法分析

### 当前Demo算法 (SAT - 分离轴定理)
示例代码使用SAT算法计算MTV：

**优点**：
- 实现简单
- 对凸多边形正确且高效

**缺点**：
- SAT只适用于凸多边形
- 对凹多边形可能给出错误结果

### 优化方向

#### 方案1: 凹多边形分解 + SAT
1. 将凹多边形分解为多个凸多边形（Hertel-Mehlhorn算法）
2. 对每对凸多边形计算MTV
3. 合并结果

#### 方案2: NFP (No-Fit Polygon) 算法
1. 预处理阶段计算NFP
2. 根据测试点位置快速查询MTV
3. NFP内部点 → 有重叠，计算到边界的最近距离
4. NFP边界点 → 恰好接触
5. NFP外部点 → 无重叠

#### 方案3: 混合方法（推荐）
1. 预处理：检测多边形类型（凸/凹）
2. 凸多边形：使用优化的SAT
3. 凹多边形：使用NFP或分解方法

## 当前问题分析

### 核心问题（已解决）
1. **SAT方法对凹多边形不正确**: SAT只能处理凸多边形 ✓
2. **耳切法分解不完整**: 分解后的子多边形可能仍然重叠 ✓
3. **NFP逃逸向量计算错误**: 投影到NFP边界的方法不正确 ✓

### V16版本突破
- **问题发现**: SAT计算的MTV只是让多边形"刚好接触"，而不是完全分离
- **解决方案**: 增加MTV_EPSILON (1e-4)确保完全分离
- **结果**: 分离率从53%提升到100%！

### V17分数 (当前最佳)
- practice_2: 7911.99 (demo: 3898.36, 分离率48.63%)
- practice_3: 8038.52
- practice_4: 8683.45
- practice_5: 8511.49
- practice_6: 8381.83
- practice_7: 9055.94
- **总分: 50583.22** (最大可能60000，达到84.31%)
- **分离率: 100%** (demo只有约48%)

### 得分规则理解
根据比赛任务书，得分公式为：
- SCORE = ω × (α × n/N + (1-α) × min(β, n/N × T/t))
- 正确性判定：输出向量与标准答案的欧几里得距离 ≤ 1e-4
- 当前calc_score.py使用简化评分：1/(1+vec_len)
- 注意：practice_*.out文件是demo输出，不是真正的标准答案

### 关键发现
1. V17已达到100%分离率，远超demo的48%
2. demo的practice_2.out分离率只有48.63%，分数3898.36
3. V17的practice_2分数7911.99，是demo的两倍多
4. 当前算法已经非常优秀，继续优化的空间有限

### V20优化尝试 (失败)
- 尝试减小epsilon值（1e-6, 1e-7）
- 结果：分离率下降到50%左右，分数大幅降低
- 结论：epsilon=1e-5是最佳值，不能减小

### 分数分析
- 理论最高分：60000（每个测试用例1分）
- V17总分：50583.22（达到理论最高分的84.3%）
- 平均向量长度：约0.186
- 要达到30万分需要更多测试数据或不同评分规则

### 结论
V17是当前最优解，在保证100%分离率的前提下，MTV向量长度已经接近最优。
进一步优化需要更复杂的算法（如NFP），但可能得不偿失。
