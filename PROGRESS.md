# 华为软件精英挑战赛 - 进度记录

## 比赛概述

### 题目名称
简单多边形重叠消除 - 最小平移向量计算

### 核心问题
给定两个简单多边形A和B，计算最小平移向量(MTV, Minimum Translation Vector)，使多边形B沿该向量平移后与多边形A完全无重叠。

### 应用场景
二维排样、工业制造中的材料利用率优化

---

## 关键技术要点

### 1. 输入输出格式
```
输入：
- 第1行: n1 n2 (多边形A和B的顶点数)
- 第2行: 多边形A的顶点坐标 (2*n1个浮点数，逆时针)
- 第3行: 多边形B的顶点坐标 (2*n2个浮点数，逆时针)
- OK
- 第4行: K (测试样本点数量)
- 后续K行: 每行一个移动向量(x, y)
- OK

输出：
- K行最小平移向量
- OK
```

### 2. 时间限制
- 预处理：1秒
- 求解：10分钟（所有测试样本点）

### 3. 精度要求
- 阈值：ε = 1e-4
- 输出保留至少5位小数

### 4. 环境限制
- Ubuntu 22.04, 1核CPU, 2GB内存
- 单文件编译
- Python只支持numpy

## 算法分析

### 当前Demo算法 (SAT - 分离轴定理)
示例代码使用SAT算法计算MTV：

**优点**：
- 实现简单
- 对凸多边形正确且高效

**缺点**：
- SAT只适用于凸多边形
- 对凹多边形可能给出错误结果

### 优化方向

#### 方案1: 凹多边形分解 + SAT
1. 将凹多边形分解为多个凸多边形（Hertel-Mehlhorn算法）
2. 对每对凸多边形计算MTV
3. 合并结果

#### 方案2: NFP (No-Fit Polygon) 算法
1. 预处理阶段计算NFP
2. 根据测试点位置快速查询MTV
3. NFP内部点 → 有重叠，计算到边界的最近距离
4. NFP边界点 → 恰好接触
5. NFP外部点 → 无重叠

#### 方案3: 混合方法（推荐）
1. 预处理：检测多边形类型（凸/凹）
2. 凸多边形：使用优化的SAT
3. 凹多边形：使用NFP或分解方法
