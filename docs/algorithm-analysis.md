# 算法分析文档

## 问题描述

给定两个简单多边形A和B，以及一个移动向量v，将多边形B按向量v移动后，计算最小平移向量(MTV)，使得移动后的多边形B与多边形A恰好消除重叠。

## 核心概念

### 1. 简单多边形
- 由平面上有限条线段组成的闭合图形
- 线段仅在端点处相交，无自交现象，且无空洞
- 顶点按逆时针方向排列

### 2. 最小平移向量 (MTV)
- 能够恰好消除两个重叠多边形重叠区域的平移向量
- 在所有满足条件的平移向量中长度最短

### 3. 临界多边形 (NFP)
- 给定多边形A和B，固定A，将B绕A环形运动一周
- B始终与A接触且无重叠
- B上参考点的运动轨迹构成NFP

## 算法方案

### 方案一：分离轴定理 (SAT) - 当前Demo实现

**原理**：
1. 对于凸多边形，如果存在一条分离轴使得两个多边形在该轴上的投影不重叠，则两多边形不相交
2. MTV的方向就是最小重叠量的分离轴方向
3. MTV的大小就是最小重叠量

**适用性**：
- ✅ 凸多边形：完全正确
- ❌ 凹多边形：可能给出错误结果

**时间复杂度**：O((n1+n2) × K)，其中n1,n2为顶点数，K为测试样本数

### 方案二：NFP算法

**原理**：
1. 预处理阶段计算NFP
2. 测试点在NFP内部 → 有重叠，MTV = 到NFP边界的最近距离
3. 测试点在NFP边界 → 恰好接触，MTV = (0,0)
4. 测试点在NFP外部 → 无重叠，MTV = (0,0)

**优点**：
- 预处理后查询效率高
- 正确处理凹多边形

**缺点**：
- NFP计算复杂
- 预处理时间可能较长

### 方案三：凹多边形分解 + SAT

**原理**：
1. 将凹多边形分解为多个凸多边形
2. 对每对凸多边形计算SAT
3. 合并所有MTV，取最优解

**分解方法**：
- Hertel-Mehlhorn算法
- Keil-Snoeyink算法

### 方案四：多边形裁剪法

**原理**：
1. 计算两个多边形的重叠区域
2. 计算重叠区域的几何中心
3. 从中心向最优方向移动

## 推荐方案

考虑到：
1. 预处理时间限制1秒
2. 测试样本可能很多（practice_1有10000个）
3. 需要处理凹多边形

**推荐混合方案**：
1. 预处理阶段：
   - 检测多边形类型（凸/凹）
   - 如果都是凸多边形，预处理SAT所需数据
   - 如果有凹多边形，计算NFP或分解

2. 查询阶段：
   - 凸多边形：使用优化的SAT
   - 凹多边形：使用NFP查询

## 精度处理

- 使用double精度
- 阈值ε = 1e-4
- 输出保留5位小数

## 性能优化策略

1. **空间换时间**：预处理存储边法向量
2. **提前终止**：检测到无重叠立即返回
3. **缓存优化**：连续内存访问
4. **并行计算**：如果允许，可并行处理多个测试样本
